# Nokia-225-RE

Reverse engineering the Nokia 225 4G (TA-1282).

## Goals

- Document the hardware
- Obtain and analyze official firmware
- Identify bootloader and download modes
- Reverse engineer the operating system
- Investigate native application support
- Determine whether custom firmware is possible
